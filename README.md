# Публикация базы данных
1. Правой кнопкой мыши по проекту Db_Task_5
2. Выбрать "Опубликовать..."
3. В появившемся окне, слева от "Подключение к целевой БД" нажать на кнопку "Изменить"
4. Выбрать куда нужно опубликовать БД (желательно на locahHost) и нажать на "ОК"
5. После ввести имя базы данных
6. Если БД была опубликована на locahHost и имя БД было задано как "Db_Task_5", тогда строку подключения менять не нужно
7. Если БД была опубликовано не на locahHost и заданное имя БД отличается, тогда в строке подключения поменять название сервера и имя БД на ваши
ПРИМЕР:
Data Source=ИМЯ_ВАШЕГО_СЕРВЕРА;Initial Catalog=ИМЯ_ВАШЕЙ_БД;Integrated Security=True
